Make a Portfolio Website

Last updated: May 4, 2020

This tutorial will take you through the process of building a simple portfolio website from scratch using Glitch. The final product will look like this:



1) Start a new Glitch project

1.1) Go to glitch.com.

1.2) Click on "New Project" and "hello-webpage".

1.3) In the box on the left, delete all of the files except index.html and assets.

1.4) Delete all of the text in the index.html file. We want to start from scratch.

2) Add two more HTML files

2.1) In the box on the left, click on "New File" and add a file named "about.html".

2.2) Add another file named "cv.html".

3. Start the "about" file

3.1 Open about.html and add body tags to the file: 
<body>
</body>
3.2 Inside the body tags, add a heading using "h1" tags: 
<body>
  <h1>About Me</h1>
</body>
3.3 Under the heading, add links to each of the three pages (Home, About Me, and CV) using "a" tags: 
<body>
  <h1>About Me</h1>
  <a href="index.html">Home</a>
  <a href="about.html">About Me</a>
  <a href="cv.html">CV</a>
</body>

4) Add a heading and links to the other two pages

4.1) Copy the code from about.html into index.html and cv.html.

4.2) Change the headings from "About Me" to "My New Website" in index.html and "CV" in cv.html.

4.3) Click on "show" at the top of the glitch window to see your website. All of the pages should look like this:

5) Add content to the "about" page

8) Find fonts from Google Fonts

8.1) Go to fonts.google.com.

8.2) Select two fonts you want to use. I will use Viaoda Libre for headings and Libre Franklin for paragraphs.

8.3) Click "view your selected families" in the upper right corner of the Google Fonts page.

8.4) Go to the "embed" tab and copy the font link.

8.5) Go to about.html and paste the font link between the "head" tags:

<script>
  function toggleOpen(name){
    var section = document.getElementById(name);
    if(section.style.display !== "block"){
      section.style = "display:block;"
    }else{
      section.style = "display:none;"
    }
  }
</script>
